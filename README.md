# 🎉 Party Effect - Interactive Celebration Page

A simple, interactive, and colorful celebration web page built with HTML, CSS, and Vanilla JavaScript.

## ✨ Features

- **Interactive Greeting:** Enter your name to receive a personalized welcome message.
- **Dynamic Background:** The background continuously changes colors in a festive party mode.
- **Confetti Animation:** Colorful confetti pieces fall gracefully from the top of the screen.
- **Fireworks Effect:** Exciting firework explosions appear across the screen using JavaScript and CSS animations.
- **Modern UI:** Glassmorphism card design with smooth transitions and hover effects.

## 🚀 How to Run Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/Rupesh72550/Party_Effect.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Party_Effect
   ```
3. Open the `index.html` file in your preferred web browser.

## 🛠️ Technologies Used

- **HTML5:** Structure of the web page.
- **CSS3:** Styling, layout, and keyframe animations (confetti, fireworks, background transitions).
- **JavaScript:** DOM manipulation, handling user input, and dynamically generating the celebration effects.

## 📁 Project Structure

- `index.html` - The main structure of the page.
- `style.css` - All styles and CSS animations.
- `script.js` - The interactive logic for the celebration effects.
